PyHadoop
========

Python based Hadoop command-line interface

How to install?
---------------
```
$> sudo pip install pyhadoop
```

What is this?
--------------
It's very samll app.
Actaully I have build it for myself to automate daily task related to hadoop.

Usage:
-------
```
pyhadoop -h     : for help
pyhadoop start  : starts hadoop
pyhadoop stop   : stops hadoop
```
